# Diagnóstico ServiciosPlus

Los servicios de suscripción dentro del ecosistema de plataformas digitales de intermediación, como Rappi o Uber, tienen como objetivo incrementar la frecuencia de compra de los usuarios, fortalecer su fidelización y maximizar el valor generado para el negocio.

El propósito de este proyecto es evaluar la efectividad de un programa de suscripción y determinar si cumple con sus objetivos estratégicos, respondiendo a las siguientes preguntas clave:

* ¿Los usuarios suscritos realizan compras con mayor frecuencia que los usuarios no suscritos?
* ¿El modelo de suscripción genera rentabilidad para la compañía?
* ¿Existen oportunidades de mejora o puntos de fricción en el proceso de compra que impacten negativamente la conversión y la experiencia del usuario?

#### 1. Identificación y corrección de inconsistencias
Se realizará un proceso de exploración y limpieza de datos para identificar inconsistencias entre los diferentes conjuntos de datos utilizados en el proyecto. Esto incluye la validación de registros, tratamiento de valores faltantes y estandarización de tipos de datos, con el objetivo de garantizar la calidad de la información utilizada para el análisis de rentabilidad.

#### 2. Evaluación de la rentabilidad del negocio
Se calcularán los principales indicadores de desempeño (KPIs) relacionados con ingresos, costos y rentabilidad para responder las siguientes preguntas:

* ¿Cuál es el ingreso total generado (Revenue)?
* ¿Cuál es el costo total de operación?
* ¿Cuánto se ha invertido en actividades de marketing?
* ¿El programa de suscripción es rentable? (Profit)

#### 3. Análisis de conversión y retención de usuarios
A través de un embudo de conversión (Conversion Funnel), se identificarán las etapas en las que los usuarios abandonan el proceso de compra. Adicionalmente, mediante un análisis de cohortes de retención, se evaluará el comportamiento de los clientes después de su primera compra para determinar qué porcentaje regresa y continúa utilizando la plataforma.

#### 4. Dashboard interactivo – Primer semestre (YTD)

Con el fin de visualizar el desempeño acumulado del año (Year-to-Date, YTD), se desarrolló un dashboard interactivo que permite monitorear métricas clave de negocio, incluyendo ingresos por mes, categoría y producto, facilitando la toma de decisiones basada en datos.

### Conclusión

Los hallazgos obtenidos permitirán identificar oportunidades de optimización en la experiencia de usuario, la estrategia de precios y el modelo de suscripción. Asimismo, proporcionarán recomendaciones orientadas a mejorar la conversión, incrementar la rentabilidad y fortalecer la satisfacción de los clientes.
